Tunnel Flow - Single Binary Distribution 
 
Files: 
- tunnel-flow.exe: Main server (includes web UI) 
- tunnel-flow-agent.exe: Agent client 
- config.yaml: Configuration file 
- ssl/: SSL certificates (if exists) 
- start.bat: Quick start script 
 
Usage: 
1. Edit config.yaml if needed 
2. Run start.bat or tunnel-flow.exe directly 
3. Open http://localhost:8080 in your browser
